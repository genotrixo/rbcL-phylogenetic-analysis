</p># rbcL Phylogenetic Analysis — Lamiaceae

## Overview
This repository contains the R code for constructing a circular phylogenetic tree based on the **rbcL** gene sequences of selected Lamiaceae species.

## Target Species (Sequenced in this study)
- *Ocimum basilicum*
- *Mentha longifolia*
- *Thymus capitatus*

## Tools & Libraries
- R (ggtree, ggplot2, ape)

## Output
- Circular phylogenetic tree with color-coded evolutionary distances
- Gold star markers (★) highlighting the three target species

## Citation
Code available at: https://github.com/genotrixo/rbcL-phylogenetic-analysis
